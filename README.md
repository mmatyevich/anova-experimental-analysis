# ANOVA Experimental Analysis

## Overview
This project applies ANOVA to assess differences between groups in experimental settings.

## Methods
- One-way and/or repeated measures ANOVA
- Post-hoc comparisons

## Key Insights
- Identified statistically significant differences between conditions
- Demonstrated interpretation of group effects in experimental data

## Tools
R (aov, tidyverse)

## Relevance
Demonstrates core statistical inference techniques used in research and health data analysis.
